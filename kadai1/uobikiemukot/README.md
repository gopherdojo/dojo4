# description

convert images included in specified directory.

# usage

~~~
$ convert [-i INPUT_FORMAT] [-o OUTPUT_FORMAT] DIRECTORY
~~~

- `-i`: input image format (default: jpg)
- `-o`: output image format (default: png)

# make subcommands

## lint

lint source code.

~~~
$ make lint
~~~

## test

run test.

~~~
$ make test
~~~

## build

create excutable binary.

~~~
$ make build
~~~

## doc
